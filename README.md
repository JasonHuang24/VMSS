## About

The Five Rings is a proposed civilization model organized around five concentric governance rings (+1 to -3). Citizens are placed in rings based on demonstrated behavior вЂ” not birth, wealth, or ideology. The system combines technoneural enforcement, a Social Trust Index, automation-funded UBI, backup vessel continuity, and consequence-bound freedom into a single coherent architecture.

The site functions as a doctrine portal: it explains the model, makes the logic legible, and provides interactive tools to explore how the system evaluates behavior.

## Key Concepts

**VMSS** вЂ” Vertical Moral Stratification System. Behavioral stratification replaces incarceration. Layer placement is a permanent environmental consequence of demonstrated conduct, not a time sentence.

**STI** вЂ” Social Trust Index. A separate ledger for non-criminal trust violations. Covers harms that fall below legal thresholds вЂ” fraud, chronic deception, harassment вЂ” and makes them socially legible without criminalizing them.

**Backup Vessels** вЂ” Periodic encrypted mind-state backups. Revival preserves continuity, not innocence. Death in -3 Terminal is final.

**Neural Diving** вЂ” Direct mind-to-mind interface technology. Audience mode (passive observation) and Pilot mode (temporary control, requires consent). The civilization's primary medium for empathy, education, and art.

**SADs** вЂ” Selective Ascension Domains. Opt-in sub-zones within +1 Sanctuary, each gated by a single measurable metric. Citizens may qualify for multiple simultaneously.
