## The S9 gap — confirmed

LP-043's trigger text is verbatim as claimed and conjunctive: review requires **both** >40% MGD membership penetration in a service category **and** measurably degraded non-member access (laws.html:796; elder-care is a named category). The Metric Gated Domain Act's blindness rule is absolute — "no civilizational registry, compiles no enumerations... deliberately blind to aggregate structure **even for administrative purposes**" (laws.html:2309). The sweep found no sensor, no self-report convention, no statistical proxy anywhere — and LP-043's own ratification record debated scope (lower-layer autonomy) without ever discussing measurement. **The statute cannot lawfully observe its own trigger condition.** Verified GAP.

## Verification-discovered finding neither model flagged: LP-025's layer scope

**LP-025 (Cross-Layer Marriage Recognition Standards) is ratified Main + Sanctuary joint only** (laws.html:2552/2558; law-polling.html:2839). Both runs cited it for marriages reaching lower layers — ChatGPT S7 (Main↔-1 elective residency) and Opus S7 (Main↔-2 VPR). Under XXVIII's own tier rule, a layer-wide regulation binds only its enacting layers. Canon supplies **no instrument** extending property/inheritance/medical-consent marriage recognition to Main↔-1/-2/-3 marriages; only the revival-identity strand carries everywhere via the federal Continuity Integrity Act ("marriages, contracts, property and layer status carry through"). Given that elective residency and VPR *exist to let families span layers*, this is a real seam both scenario generators walked straight across without noticing. **New queue item.**
